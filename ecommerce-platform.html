Clae Ecommerce


To explore and learn more about Cloud Engineering and Solution Management, our team selected Google Cloud Platform (GCP) as the cloud provider for our Ecommerce online store application.

Description

The cloud native application features an end-to-end solution, consisting of the full stack application, frontend cloud infrastructure, backend load balancer, containerized services in Google Kubernetes Engine (GKE), scheduled data backup with Cloud Scheduler and Cloud Functions, monitoring and logging with GCP Operation Suite, Slack integration for alerting and CI/CD pipeline with Github Actions.

I was responsible for designing and building the frontend application with React.js, payments microservice and Stripe API integration, order fulfilment workflow with Google Cloud Pub/Sub and frontend cloud infrastructure (CDN, DNS, etc).

Technology

Takeaways

It was a really enriching and rewarding experience. I learnt a lot about Cloud Computing and understanding how each component of the solution ensures the availability, reliability and scalability of the services.
