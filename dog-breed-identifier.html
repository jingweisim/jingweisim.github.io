Telegram ChatBot Dog Breed Identifier


Hack&Roll 2021 was a 24 hours Hackathon that spanned from 8 to 9 Jan 2021.

As a fun side project to explore and learn more about Machine Learning and Software Development, our team decided to develop a Dog Breed Classification model and host it on a Telegram Chatbot.

A quick run through of the Bot:

Description

The Telegram Chatbot allows users to submit an image of a dog and on the backend, the image is being processed and passed into the Convolutional Neural Network model to predict the class and classify the breed. If users were to send a text-based message to the Telegram Chatbot, the Bot will reply with an image or gif of a dog instead while it awaits a dog image from the user.

Process

Throughout this learning experience, we had the opportunity to source for a suitable dataset for our Machine Learning model, perform Exploratory Data Analysis to identify any missing data and execute data imputation if necessary. We decided on the Stanford Dogs Dataset as it features 20,000 images of 128 classes of dog breeds that is sufficient for model training and contains a diverse selection of dog breeds that we adore.

We performed data preparation on the image dataset, such as image resizing, image transformation into numpy arrays and data input pipeline building using TensorFlow. Next, we underwent model selection, building, training and evaluation. Through the model selection phase, we selected MobileNetV2 feature extractor headless model from TensorFlow Hub that was trained on ImageNet datasets and applied dense layers to classifying the image into the 128 dog breed classes.

Evaluation

During model training, we froze the feature extractor layer so that only the weights in the classification layers are being updated. Despite the high training accuracy of 99% on 10 epochs, there was a drop in validation accuracy (84%) which could be showing signs of overfitting as the training loss and validation loss started to diverge as early as the second epoch.

The model performed well on certain dog breeds that had distinct features such as Poodles that had brown and curly fur. On the other hand, the model has a hard time discerning between similar looking dog breeds such as Japanese Spitz and Samoyed.

Technicalities

  • • Telegram Bot API
  • • CNN Model using TensorFlow
  • • Transfer Learning using MobileNetV2
  • • 99% Training Accuracy | 84% Validation Accuracy
  • • Deployed to Heroku
