TelegramBot Dog Breed Identifier


Hack&Roll 2021 was a 24 hours Hackathon that spanned from 8 to 9 Jan 2021.

As a fun side project to explore and learn more about Machine Learning and Software Development, our team decided to develop a Dog Breed Classification model 🐕 and host it on a Telegram Bot.

Description

The Bot allows users to submit an image of a dog and the Telegram bot will run an inference on the image using a trained MobileNetV2 Convolutional Neural Network model to predict the breed of the dog.

Technology

  • • Telegram Bot API
  • • Transfer Learning using TensorFlow Hub's MobileNetV2 CNN model
  • • Trained on Stanford Dogs Dataset (20,000 images)
  • • 84% Test Accuracy
  • • Deployed to Heroku

Takeaways

It was a really fun and rewarding learning experience. It was an enriching hackathon that opened my perception to the possibilities and capabilities of Deep Learning and Computer Vision.
