Final Year Project - StockItUp


Our team collaborated with Stemly, an AI Software-as-a-Service (SaaS) company for our Final Year Project to predict the demand of retail products in the Fast Moving Consumer Goods (FMCG) industry.

Demo with mock data

Description

The web application features a frontend dashboard that displays key insights about the transaction data and allows users to generate demand forecasts and shelf replenishment timing predictions using machine learning.

The backend system features an API application, data/ML pipelines and other containerized services, allowing users to run automated machine learning model training and data preprocessing tasks asynchronously.

The backend services communicate via events with Redis Pub/Sub and these events from the Airflow server are communicated to the frontend using WebSockets to allow for real time notifications to the dashboard for long running tasks. Model training pipeline conducts hyperparameter tuning with Optuna and deploys the best performing model.

Technology

  • • Business Metrics Dashboard
  • • Demand Forecasting with Temporal Fusion Transformer
  • • Multi-label Shelf Replenishment with Machine Learning
