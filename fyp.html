Final Year Project


Our team collaborated with Stemly, an AI Software-as-a-Service (SaaS) company for our Final Year Project to predict the demand of retail products in the Fast Moving Consumer Goods (FMCG) industry.

Demo with mock data

Description

The web application features a frontend dashboard and a backend machine learning system that is built using data pipelining tool and other distributed services, allowing users to run machine learning tasks asynchronously. The backend services exchange events via Redis Pub/Sub and these events are communicated to the frontend using WebSockets to allow for real time updates to the dashboard.

Technology

  • • Business Metrics Dashboard
  • • Data cleaning & preprocessing pipeline
  • • Machine Learning model retraining pipeline
  • • Machine Learning inference
