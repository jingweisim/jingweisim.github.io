Stock Price Platform


Our team collaborated with an AI Software-as-a-Service (SaaS) company for our Final Year Project to predict the demand of retail products in the Fast Moving Consumer Goods (FMCG) industry.

Description

The web application features a Vue.js and Chart.js frontend UI that displays the technical indicators charts using Yahoo Finance API for real time data retrieval and Python Pandas for generation of the indicator.

The Flask backend system features a microservices architecture to support the processing of technical indicators generation (MACD, RSI, etc), efficient frontier computation for portfolio optimization and portfolio price forecasting with XGBoost Regressor. Price is used as the main feature in the forecasting and multi-step forecasting is achieved using autoregressive feedback loops.

Technology

  • • Technical Indicators Analysis Charts
  • • Portfolio Optimization with Efficient Frontier computation
  • • ETF price prediction with Machine Learning
